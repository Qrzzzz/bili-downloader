# Code of Conduct

## 我们的承诺

本项目希望保持专业、友善、尊重事实的协作环境。参与者应尊重不同背景和经验水平的贡献者，并围绕问题本身进行讨论。

## 期望行为

- 使用清晰、尊重的语言。
- 聚焦技术事实、复现步骤和可验证结果。
- 接受维护者对合规、安全和项目范围的边界判断。
- 在提交日志、截图或示例时主动脱敏。

## 不接受的行为

- 人身攻击、骚扰、歧视或威胁。
- 发布他人隐私、账号信息、Cookie 或登录态。
- 要求或贡献绕过付费、会员、DRM、地区限制、风控或访问权限限制的功能。
- 鼓励侵犯版权或违反平台规则的使用方式。

## 执行

维护者可以编辑、隐藏或关闭违反本准则的 issue、PR、评论或讨论，并在必要时限制相关账号继续参与。
